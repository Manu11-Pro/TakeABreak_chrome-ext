# TakeABreak_chrome-ext

This is a Chrome extension which reminds the user to take a break!Taking a break is essential if you want to be productive and also for the users mental and physical health.

## Installation

- Go to [Releases in Github](https://github.com/Manu11-Pro/TakeABreak_chrome-ext/releases/tag/v1.0.0)
- Download TakeABreak_chrome-ext.zip
- Unzip the TakeABreak_chrome-ext.zip in your computer
- Open Chrome and go to chrome://extensions
- Turn on developer mode
- Click Load unpacked and select the folder you unzipped.

## Notes on What this Extension is

- The project is just a simple Take A break ! chrome extension as its name suggests.

- It only gives a notification.

- I personally believe that to make a productivity and health boost extension, there should be minimal bloat.

- It will give you a notification after 45 minutes to take a 15 min break.
